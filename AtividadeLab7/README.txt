A Federação Mineira de Futebol contratou você para escrever um programa para fazer uma estatística do resultado de vários JOGOS. Escreva um algoritmo para ler o número de gols marcados pelo Atlético e o número de gols marcados pelo Cruzeiro em um duelo, imprimindo o nome do time vitorioso ou a palavra EMPATE. Logo após escrever a mensagem “Novo DUELO 1.Sim 2.Não?” e solicitar uma resposta. Se a resposta for 1, o algoritmo deve ser executado novamente solicitando o número de gols marcados pelos times em uma nova partida, caso contrário deve ser encerrado imprimindo:

(a) Quantos jogos fizeram parte da estatística.
(b) O número de vitórias do Atlético.
(c) O número de vitórias do Cruzeiro.
(d) O número de Empates.
(e) Uma mensagem indicando qual o time que venceu o maior número. de duelos (ou não houve vencedor).
