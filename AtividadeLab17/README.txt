Estoque de Produtos - Fase 3

Nesta atividade prática iremos finalizar o desenvolvimento uma aplicação para gerenciar o estoque de produtos de uma empresa.
Nesta terceira fase você deve adicionar o recurso de salvar os dados do sistema em um arquivo Texto.

Você deve ler o arquivo e carregar os dados salvos assim que o sistema for iniciado e deve atualizar o arquivo 
toda vez que o sistema for encerrado.
