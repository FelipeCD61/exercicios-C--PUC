Finalmente estamos prontos para fazermos nosso primeiro programa realmente funcional. 
O objetivo é construir um programa que realiza diversas operações matemáticas. 
Todas as operações deverão ser implementadas como funções. Algumas destas funções já estão disponíveis na biblioteca MATH.
Utilizando o conceito de funções, construa um programa com os seguintes requisitos: O programa possui a seguinte tela inicial
 

O usuário deve digitar uma das opções do menu. O programa não deve aceitar do usuário opções inválidas. 
A cada opção válida selecionada, o programa deve pedir os parâmetros necessários para a realização da função escolhida e deve retornar o resultado da função avaliada. Ao final da execução da função, o usuário deverá pressionar uma tecla para retornar à tela inicial. O programa só interromperá a execução quando o usuário pressionar a tecla ‘x’.
Cada uma das 7 funcionalidades do programa acima deverá ser implementada usando-se o conceito de funções.
