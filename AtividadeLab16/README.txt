Estoque de Produtos - Fase 2

Nesta atividade prática iremos continuar a desenvolver uma aplicação para gerenciar o estoque de produtos de uma empresa. 
Nesta secunda fase iremos adicionar a classe (Class) Produto ao sistema.

A classe Produto deve conter as seguintes informações:

* Código do Produto
* Nome do Produto
* Preço do Produto
* Quantidade de Produtos

Após atualizar todo o sistema para utilizar aa estrutura Produto, você deve atualizar a funcionalidades para permitir:

1. Cadastrar um novo produto
2. Listar todos os produtos cadastrados
3. Editar dados de um produto
4. Pesquisar um produto por nome
5. Pesquisar um produto por código
6. Listar todos os produtos com estoque inferior a 10
7. Apagar um produto
8. Sair do Programa
